# RAPID Tracker

## Overview
Rapid Alert & Protection Incident Dispatch (RAPID) Tracker is a tourist safety monitoring system designed for high-risk areas.

## Features
- Real-time Tourist Monitoring
- Emergency Alert System
- Secure Verification
- Google Maps Integration
- Blockchain-based Security

## Tech Stack
- React.js
- Flutter
- Node.js
- Firebase
- Hardhat
- Ganache
- Google Maps API

## Problem Solved
Provides faster emergency response and improved tourist safety in remote areas.
